OSINT Ghost V1

OSINT Ghost V1 es una tool hecha en Python con varias funciones de OSINT, red y utilidades básicas. Incluye logs y distintas herramientas integradas para análisis de información pública.

Funciones:
- Username search (búsqueda de usernames en redes)
- Username generator
- IP lookup
- DNS lookup
- Port scanner
- Website information extractor
